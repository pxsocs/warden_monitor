# INSTALLATION

```bash
git clone https://github.com/pxsocs/warden_monitor
cd warden_monitor
python3 -m venv myenv
source myenv/bin/activate
python3 -m pip install -r requirements.txt
```

Then to run:

```bash
source myenv/bin/activate
cd src
python3 warden
```

Upgrade:

```bash
git pull origin master
```
